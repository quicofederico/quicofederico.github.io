Mis Proyectos

  • Examen en Java

    Desarrollar un formulario de examen en Java que seleccione aleatoriamente 10 preguntas de un banco total de al menos 60 preguntas. Evaluar la calificación del usuario al finalizar el examen y mostrar las preguntas incorrectas. Implementar una funcionalidad que registre las preguntas contestadas si el usuario hace clic fuera del formulario.

  • Instrucciones para la Actividad de Programación en Java

    Crear una aplicación en Java que permita gestionar tareas con diferentes prioridades y duraciones. La aplicación debe contener un formulario para ingresar tareas, y tres tablas para visualizar las tareas según su prioridad. Además, se debe poder marcar las tareas como cumplidas y visualizar cuáles ya han sido completadas.

  • Archivo Secuencial Indexado

    Desarrollar un programa en Java que administre archivos por tamaño, tipos y fecha.

  • Cajero Automático Interactivo en Java

    Desarrollar una aplicación que simule un cajero automático interactivo con funciones de retiro, consulta de saldo, y más.
